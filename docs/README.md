# KojeomGameEngine Documentation

KojeomGameEngine의 기술 문서와 개발 가이드라인을 관리합니다.

## Directory Structure

```text
docs/
  getting-started.md           # 시작하기 가이드
  rules/
    AIAgentRules.md            # 코딩 컨벤션, 안정성, 유지보수, 보안 가이드라인
  feature/
    architecture.md            # 엔진 아키텍처 개요
    api-reference.md           # EngineInterop API 레퍼런스
    samples-guide.md           # 샘플 프로젝트 가이드
    assets/                    # 에셋 시스템
    audio/                     # 오디오 시스템
    debugui/                   # 디버그 UI
    editor/                    # 에디터
    input/                     # 입력 시스템
    physics/                   # 물리 시스템
    renderer/                  # 렌더러 (DirectX11Renderer.md, rendering-pipeline.md)
    scene/                     # 씬 시스템
    serialization/             # 시리얼라이제이션
    ui/                        # UI 시스템
    utils/                     # 유틸리티
```

## For AI Agents

- 코딩 가이드라인: `docs/rules/AIAgentRules.md`
- 구현 아키텍처: `work/architecture/`
